## Python
1. `pip install -r requirements.txt`
2. `chmod +x unit-tests.sh`
3. `./unit-tests.sh`
4. `python3 -m venv venv`
5. `./venv/bin/activate`

# Migration
1. `flask init db`
2. `flask db migrate`
3. `flask db upgrade`
